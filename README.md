# xmrig-auto
Installs on Raspberry Pi board to setup mining crypto.

## setup to clone file
```bash
sudo apt-get install git -y
```
```bash
git clone https://github.com/AKABigDinner/xmrig-auto.git
```
```bash
cd xmrig-auto
```
```bash
sudo chmod +x install-xmrig.sh
```
```bash
./install-xmrig.sh
```
```bash
cd xmrig/build
```
```bash
nano config.json
```
